# 硒相关基因表达分析项目

## **项目简介**
本项目分析硒相关基因在人体器官和细胞中的表达模式，揭示其生理生物学含义以及与疾病/生理学过程的关联。

## **硒蛋白基因家族**

### **GPX家族（谷胱甘肽过氧化物酶）**
- **GPX1**: 清除 H₂O₂ 和有机氢过氧化物，肝脏、肾脏、大脑高表达
- **GPX2**: 肠道特异性抗氧化酶，主要在胃肠道表达
- **GPX3**: 血浆抗氧化酶，肾脏合成后分泌到血浆
- **GPX4**: 磷脂过氧化物酶，防止铁死亡，肝脏、肾脏、大脑高表达

### **TXNRD家族（硫氧还蛋白还原酶）**
- **TXNRD1**: 抗氧化酶，维持氧化还原状态，肝脏、甲状腺、心脏高表达
- **TXNRD2**: 线粒体特异性抗氧化酶，心脏、肌肉高表达
- **TXNRD3**: 睾丸特异性抗氧化酶，主要在睾丸表达

### **SELENO家族**
- **SELENOP**: 硒转运蛋白，肝脏合成，分布到血浆和各个器官
- **SELENOS**: 炎症调节蛋白，ER应激响应，肝脏、胰腺高表达
- **SELENON**: 骨骼肌功能蛋白，钙离子调控，骨骼肌、心脏高表达
- **SELENOK**: 免疫调节蛋白，主要在免疫细胞表达
- **SELENOT**: 细胞保护蛋白，主要在肾脏和大脑表达
- **SELENOM**: 细胞增殖调节蛋白，肝脏和大脑高表达

### **硒代谢相关基因**
- **SELENBP1/SELENBP2**: 硒结合蛋白，硒储存与转运
- **TRPM6/TRPM7**: 镁和硒离子通道，硒吸收调控

## **器官表达模式**

### **高表达器官**
- **肝脏**: GPX4, TXNRD1, SELENOP, SELENBP1（硒代谢中心）
- **肾脏**: GPX3, GPX4, SELENOP, SELENOT（硒排泄器官）
- **大脑**: GPX4, SELENOM, SELENOT（抗氧化保护）
- **甲状腺**: TXNRD1, TXNRD2（硒依赖酶）

### **器官特异性表达**
- **GPX2**: 胃肠道特异性
- **TXNRD3**: 睾丸特异性
- **SELENON**: 骨骼肌特异性
- **SELENOK**: 免疫细胞特异性

## **疾病关联**

### **癌症**
- GPX4 调节铁死亡通路，影响肿瘤细胞生存
- TXNRD1 抗氧化作用，保护癌细胞
- SELENOP 硒转运异常，影响肿瘤硒水平

### **神经退行性疾病**
- GPX4 脑细胞抗氧化保护
- SELENOM 神经元增殖调节
- SELENOT 神经元细胞保护

### **心血管疾病**
- GPX3 血浆抗氧化酶，防止血管损伤
- TXNRD1/TXNRD2 心肌细胞抗氧化保护
- SELENON 心肌细胞功能保护

### **自身免疫疾病**
- SELENOS 炎症调节，控制自身免疫反应
- SELENOK 免疫调节，平衡免疫应答
- GPX1/GPX2 抑制氧化应激，减少炎症损伤

## **生物学功能**

### **抗氧化功能**
- GPX家族清除自由基，保护细胞免受氧化损伤
- TXNRD家族维持细胞内氧化还原状态，防止氧化应激

### **硒代谢与转运**
- SELENOP是硒的主要转运蛋白，维持硒平衡
- SELENBP1/SELENBP2是硒结合蛋白，参与硒储存
- TRPM6/TRPM7是硒离子通道，调控硒吸收

### **免疫调节功能**
- SELENOS调节炎症反应，控制免疫细胞功能
- SELENOK参与T细胞的免疫调节
- GPX家族保护免疫细胞免受氧化损伤

### **细胞保护功能**
- SELENOT防止异常细胞凋亡，保护细胞免受损伤
- SELENON调节肌肉细胞的钙离子平衡，保护肌肉功能
- SELENOM调节细胞增殖，防止增殖异常

## **数据来源**
- GTEx数据库: 54个器官组织的RNA-seq表达数据
- Human Protein Atlas: 器官特异性蛋白质表达数据
- TCGA数据库: 癌症组织表达数据（33种癌症）
- Human Cell Atlas: 单细胞RNA-seq数据
- PubMed文献: 硒相关的单细胞研究文献

## **分析方法**
- 表达谱分析: 使用limma、DESeq2进行基因表达分析
- 通路分析: 使用KEGG、Reactome数据库进行通路富集
- 网络分析: 使用STRING数据库构建蛋白质相互作用网络
- 相关性分析: 计算基因表达与疾病的相关性
