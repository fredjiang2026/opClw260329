## 🛠️ 使用方法

### 1. 下载所有文件
```bash
# 使用wget下载全部文件
wget -r -np https://your-server.com/path/to/project/

# 下载特定类型的文件
wget -r -np -R "index.html*" --accept="*.csv" https://your-server.com/path/to/project/
wget -r -np -R "*.csv" --accept="*.png,*.jpg" https://your-server.com/path/to/project/
wget -r -np -R "*.csv,*.png" --accept="*.html" https://your-server.com/path/to/project/
```

### 2. 查看分析结果
1. 打开 `data_download.html` 查看下载中心
2. 查看 `splicing_analysis.html` 进行可变剪切分析
3. 查看 `images/` 文件夹中的可视化图片
4. 查看 `data/` 文件夹中的多序列比对和PASA分析结果

### 3. 分析工具
- 多序列比对：使用MUSCLE软件
- 可变剪切分析：使用rMATS和SUPPA2
- 基因结构分析：使用PASA
- 可视化：使用rmats2sashimiplot, MVIEW

## 📊 数据格式说明

### CSV文件格式
格式：基因 | 组织/癌症类型 | 表达值 | 统计参数

### FASTA文件格式
```fasta
>GPX4-001 (含Sec型)
氨基酸序列...
```

### Clustal文件格式
标准Clustal格式，包含序列比对结果。

### GFF文件格式
PASA输出格式，包含基因结构和功能注释。

## 🔬 分析内容

1. **可变剪切分析**：使用rMATS和SUPPA2检测SE, MXE, A3SS, A5SS, RI事件
2. **多序列比对**：使用MUSCLE比对异构体序列
3. **基因结构分析**：使用PASA分析外显子-内含子结构差异
4. **Sec位点分析**：识别硒代半胱氨酸(UGA)位置
5. **可视化分析**：基因结构图、Sashimi plots、多序列比对图
